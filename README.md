This is a simple and clean QR Code Generator built using HTML, CSS, and JavaScript.
Users can enter any text or URL, and the app instantly generates a scannable QR code using a free public API.

## Live Demo -  https://wasim2934.github.io/QR-code-generator/

## Features

-  Generate QR Codes instantly
-  Accepts both text and URL
-  Smooth animation when QR code appears
-  Input validation
-  Red border alert when field is empty
-  Responsive & clean UI
